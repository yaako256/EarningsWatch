# debug.py コマンド仕様

## 実行形式
```bash
python3 scripts/debug/debug.py <command> [options]
````

## コマンド一覧

| コマンド     | 用途          | 必須引数     | 出力            |
| -------- | ----------- | -------- | ------------- |
| `list`   | 一覧ページ取得のモック | `--page` | ダミー記事一覧(JSON) |
| `detail` | 個別記事取得のモック  | `--url`  | ダミー記事詳細(JSON) |

## list
一覧ページ相当のダミーデータを取得する。

### 実行例
```bash
python3 scripts/debug/debug.py list --page 1
```

### 引数
| 引数         | 説明        |
| ---------- | --------- |
| `--page N` | 取得対象ページ番号 |


---
## detail
個別ページ相当のダミーデータを取得する。

### 実行例
```bash
python3 scripts/debug/debug.py detail --url https://example.com/debug/detail/1
```

### 引数
| 引数          | 説明       |
| ----------- | -------- |
| `--url URL` | 詳細ページURL |
