# 通知方法メモ

# Discord
既知の通り、WebhookURLでできる。
あるサーバのチャンネルでそれを作り、設定できる。
やったことある通り、http通信でいける

# Slack
Discordと同じようにできるらしい。
`Incoming Webhook`というのがある。
それを使って送信しようと思う。

また、Embedとしたものとして`Block Kit`があるらしい。
これを使ってリッチなテキストとして送信する。

DiscordのEmbedのColorのような、送信formatを変えないまま、グループ毎に設定できるようなものが`Block Kit`には無さそうであるため、一旦グループ名設定はWebhookURLののみとする。
Discord/Slack両方で、将来拡張として送信フォーマットを複数指定できるようにしようかと思う。
